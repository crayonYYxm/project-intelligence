让 AI 先读懂项目，再开始写代码。

Project Intelligence 把代码库里的规范、组件、接口、服务、业务流程、spec、plan、debug context 和质量检查沉淀成项目知识层。后续需求、bug 排查、review、组件/API 查询和任务维护都会优先引用这些项目事实。

它不是另一个图谱引擎，而是项目上下文层。

GitNexus 负责符号级调用链和影响面，Understand-Anything 负责架构和业务域理解。Project Intelligence 把这些信息与项目规范、质量检查、复用约束组织成 agent 每次工作前都能读取的事实层。

01 / init

初始化项目知识

生成 `.project-intel/`，提取规范、组件、Hooks、API、服务、路由、配置和质量命令，并检查可选增强工具。

02 / reuse

先复用，再实现

需求执行前检索已有组件、Hook、工具函数、服务和业务模式，减少重复造轮子。

03 / debug

系统化 bug 排查

把报错、复现、最近变更、调用链、工作样例和质量命令整理成 debug context，先找根因再修复。

04 / quality

接入已有检查

识别 ESLint、Stylelint、Prettier、tsc、vue-tsc 和项目脚本，把检查结果写入报告。

05 / backend

识别非常规后端入口

通过框架适配器、AST/调用模式和项目自定义规则识别 API、RPC、消息和任务入口。

06 / lifecycle

需求到维护闭环

支持脑暴、spec、plan、任务影响分析和执行后的知识库维护；真实 Git hook 需要显式启用。

日常不用记命令，直接说你要做什么。

CLI 是底层工具，自然语言是日常入口。agent 自动决定该刷新知识库、查组件、跑质量检查、生成 spec/plan，还是维护任务后的项目沉淀。

prepare

初始化

“初始化这个项目规范”会检查工具、生成标准、知识库、图谱摘要和报告。

spec

需求前置

“帮我写这个需求的 spec/plan”会查组件、Hook、API、服务和业务模式。

check

质量检查

“帮我检查前端质量”会汇总 lint、type、style、format 和冗余候选。

debug

问题定位

“帮我查这个 bug”会先生成 debug context，再按证据、复现、根因、假设验证推进。

maintain

执行后维护

“这个任务做完了，维护知识库”会 refresh、check 并写维护报告。

沉淀的是团队能复用的事实。

manifest

记录项目技术栈、Git 状态、图谱来源和扫描时间，是每次任务的入口。

standards

沉淀硬性规范、推断规范和候选规则，避免 agent 只凭临时上下文发挥。

knowledge

结构化保存组件、Hooks、API、Service、DTO、Entity、路由和配置。

reports

输出初始化、工具检查、debug context、任务影响、冗余、质量检查和 review 报告，方便团队审查和演进。

lifecycle

保存 specs、plans、maintenance 和可选 hook 模板，让任务执行前后都有项目级记录。

把项目上下文变成默认工作方式。

Project Intelligence 同时支持 Claude Code 和 Codex（CLI / 桌面端）。
将 https://github.com/crayonYYxm/project-intelligence.git 添加为 marketplace 仓库后即可安装。初始化会提示缺失工具并给出选项，真实 Git hook 需要显式启用。

Claude Code

添加 marketplace 仓库：

/plugin marketplace add crayonYYxm/project-intelligence

安装插件：

/plugin install project-intelligence@project-intelligence

Codex CLI

添加 marketplace 仓库：

codex plugin marketplace add crayonYYxm/project-intelligence

安装插件：

codex plugin install project-intelligence@project-intelligence

Codex 桌面端

  1. 打开 Codex 桌面端，进入 设置（Settings）
  2. 左侧导航选择 插件（Plugins）
  3. 点击右上角 添加 Marketplace（Add Marketplace）
  4. 在弹出的输入框中填入 crayonYYxm/project-intelligence
  5. 确认后，marketplace 列表中会出现 Project Intelligence
  6. 点击 安装（Install），等待安装完成即可
